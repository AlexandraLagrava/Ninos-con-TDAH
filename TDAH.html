¿Que es TDAH?

El TDAH es un trastorno del neurodesarrollo, las personas que sufren de este transtorno pueden tener algunos
problemas para prestar atencion, contolar conductas impulsivas o ser demasiados activas. Para confirmar que
un niño tiene TDAH, es un proceso de varios pasos, no hay un unico examen para diagnosticarlo.

Carcateristicas

  1. Falta de atencion
  2. Hiperactividad
  3. Impulsividad
  4. Cratividad
  5. Memoria
